# Installation Instructions

### Clone Repository

```bash
git clone https://github.com/captainawais/frontend-internship-2026.git
```

### Open Project Folder

```bash
cd frontend-internship-2026
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Open Browser

```text
http://localhost:5173
```
