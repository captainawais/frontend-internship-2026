# 🌦 Weather Info Dashboard

A modern Weather Dashboard built with React and OpenWeather API.

## Features

* Search weather by city name
* Auto detect current location weather
* Real-time weather updates
* Loading state handling
* Error handling
* Weather icons
* Temperature display
* Humidity information
* Wind speed details
* Feels like temperature
* Search history (Last 5 cities)
* Local Storage support
* Responsive Design
* Modern Glassmorphism UI

## Technologies Used

* React JS
* Vite
* JavaScript
* CSS3
* OpenWeather API

## API Features

* Current Weather Data
* City Search
* Geolocation Weather
* Dynamic Weather Icons

## Learning Outcomes

* API Integration
* Async/Await
* Fetch API
* React Hooks
* State Management
* Error Handling
* Loading States
* Local Storage
* Responsive Design

## Installation

```bash
npm install
npm run dev
```
